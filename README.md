# Video Game Market Analysis

This project started as a university course assignment analyzing the global video game market using Python. It explores sales trends across genres, platforms, and regions, applying data analysis and machine learning techniques. The project will be further developed as a personal pet project.

---

## Project Overview

- Aggregate and visualize sales data by genre and region  
- Analyze temporal sales trends with smoothing  
- Cluster genres based on sales performance using KMeans  
- Perform correlation analysis between regional sales  
- Forecast sales for the Action genre using ARIMA time series modeling  

---

## Technologies Used

- Python 3.x  
- pandas  
- matplotlib  
- seaborn  
- scikit-learn  
- statsmodels
